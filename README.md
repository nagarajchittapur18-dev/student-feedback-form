# Student Feedback Form System

This is a simple student feedback form project developed using:
- HTML
- CSS
- JavaScript
- PHP

## Features
- Collect student feedback
- Client-side validation using JavaScript
- Server-side processing using PHP
- No database used

## How to Run
1. Install XAMPP
2. Start Apache server
3. Place project in `htdocs`
4. Open `http://localhost/student_feedback/index.html`

## Note
PHP projects cannot run on GitHub Pages or Netlify.
